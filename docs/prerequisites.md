# Prerequisites

Attendee-facing. Finalized in Phase 6.

- A Claude Code subscription (attendees bring their own).
- Cluster access is provided by the presenter. No local Kubernetes setup required.
- A modern web browser. Cluster access is delivered via browser.
- Working familiarity with Kubernetes, Helm, and GitOps (Argo CD or Flux experience helps).

No prior Backstage experience required.
