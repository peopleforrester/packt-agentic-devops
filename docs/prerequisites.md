# Prerequisites

What to bring before the workshop.

## Required: a Claude Code subscription

You bring your own Claude Code subscription. The workshop is built, rehearsed, and tested entirely with Claude Code, and it is the only supported tool.

## Alternatives (may work, untested)

Other agentic coding CLIs may work if they run headless in the browser terminal with a permissioned (non-auto-approve) mode. As of June 2026 these exist and function in principle: OpenAI Codex CLI, GitHub Copilot CLI, Google Antigravity CLI, Amazon Kiro CLI, opencode, Goose, and Cursor CLI. None are tested in this workshop. If you bring one, you are on your own, and you can always fall back to watching the live build and the take-home repo.

If you do bring an alternative: avoid Gemini CLI (retired for personal plans) and Amazon Q Developer CLI (new signups closed); bring Antigravity CLI or Kiro CLI instead. Cursor CLI cannot use the in-cluster model.

## The rest

- Cluster access is provided in the browser. No local Kubernetes setup required.
- A modern web browser.
- Working familiarity with Kubernetes, Helm, and GitOps (Argo CD or Flux experience helps).

No prior Backstage experience required.
