# Agentic DevOps with Claude

An AI-native Internal Developer Platform, built live with Claude Code as the platform engineer, on Amazon EKS. Companion repo for the Packt workshop on July 23, 2026.

This README is the attendee-facing front door and is written last, in Phase 6. For now it points at what already exists.

## Where to start

- `docs/build-spec.md` is the full build spec (the plan for this repo).
- `docs/research-findings-june-2026.md` has the verified component versions and the facts behind the design.
- `docs/workshop-overview.md` and `docs/packt-event-listing.md` describe the workshop.
- `components.yaml` is the pinned component set.
- `versions.lock.md` is the quick version lookup.

## Status

Scaffolding stage. The repo structure, the component manifest, the Claude Code governance kit, and the CI pinned-version check exist. The platform manifests, provisioning automation, runbook, and prompt library are built phase by phase per the build spec.
